# C - Pointers, Arrays and Strings

This project is part of the Holberton School Low Level Programming curriculum.

## Description
This project introduces the concepts of:
- Pointers
- Arrays
- Strings

Students learn how to manipulate memory addresses, work with arrays, and handle strings in the C programming language.

## Learning Objectives
At the end of this project, you should be able to explain:

- What pointers are and how to use them
- What arrays are and how to use them
- The difference between pointers and arrays
- How to manipulate strings
- Memory addresses and pointer operations
